# Contributing / 参与贡献

This project welcomes contributions and suggestions.
本项目欢迎各类贡献与建议。

Most contributions require you to agree to a Contributor License Agreement (CLA) declaring that you have the right to, and actually do, grant us the rights to use your contribution.
大多数贡献都要求你同意签署贡献者许可协议（CLA），以声明你有权并且确实授予我们使用你所提交内容的相关权利。

For details, visit https://cla.microsoft.com.
详情请访问 https://cla.microsoft.com 。

When you submit a pull request, a CLA-bot will automatically determine whether you need to provide a CLA and decorate the PR appropriately (for example, with a label or comment).
当你提交 Pull Request 时，CLA 机器人会自动判断你是否需要提供 CLA，并为该 PR 添加相应提示（例如标签或评论）。

Simply follow the instructions provided by the bot.
按照机器人提供的说明操作即可。

You will only need to do this once across all repositories using our CLA.
在所有使用该 CLA 的仓库中，你通常只需要完成一次。

This project has adopted the [Microsoft Open Source Code of Conduct](https://opensource.microsoft.com/codeofconduct/).
本项目采用了 [Microsoft 开源行为准则](https://opensource.microsoft.com/codeofconduct/)。

For more information see the [Code of Conduct FAQ](https://opensource.microsoft.com/codeofconduct/faq/) or contact [opencode@microsoft.com](mailto:opencode@microsoft.com) with any additional questions or comments.
如需更多信息，请参阅 [行为准则常见问题](https://opensource.microsoft.com/codeofconduct/faq/)，或通过 [opencode@microsoft.com](mailto:opencode@microsoft.com) 联系相关团队提出问题或反馈。
